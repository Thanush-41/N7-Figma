# N7 — Modern Banking Frontend

Pixel-perfect responsive implementation of the N7 banking landing page.

**Live:** https://n7-figma.vercel.app

## Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS 3.4
- Framer Motion 12
